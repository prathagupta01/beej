# beej

Beej (meaning "Seed" in Hindi) is a project designed to make crop production scalable and profitable to farmers. It focuses on providing a clean, modular structure for AI assist for crop production , crop failure and recommanation on fertilizers, crop and many more.

## 🚀 Features
Modular Architecture: Built with high scalability in mind, separating business logic from UI components.

Secure Authentication:Implemented using JWT for robust user data protection.

RESTful API Integration: Leverages Node.js and Express to handle complex data-driven requests efficiently.

AI assistent : made with llm models (hugging faces) with flask and python.

## 🛠️ Tech Stack

Frontend: HTML5, CSS3, JavaScript.<br>
Backend: Node.js, Express.js.<br>
AI Orchestration: Real-time streams via hugging faces.
